# kiro-blast

a bomberman-style retro game built during the aws re:invent 2025 workshop

## about

this game was created at the workshop:
**game on: build a retro adventure game in 120 minutes (DVT402)**
during aws re:invent 2025 in las vegas

## how to play

- use arrow keys to move kiro around the arena
- press spacebar to place vibes (bombs)
- destroy all enemies to complete the level
- avoid explosions and enemies
- collect points by destroying bricks and enemies

## features

- bomberman-style gameplay with a kiro twist
- retro pixel art aesthetic
- smooth animations and responsive controls
- multiple lives system
- score tracking

## running the game

simply open `game.html` in your web browser to start playing!
