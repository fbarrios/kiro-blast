## how to play

- use arrow keys to move kiro around the arena
- press spacebar to place vibes (bombs)
- destroy all enemies to complete the level
- avoid explosions and enemies
- collect points by destroying bricks and enemies

## features

- bomberman-style gameplay with a kiro twist
- retro pixel art aesthetic
- smooth animations and responsive controls
- multiple lives system
- score tracking
